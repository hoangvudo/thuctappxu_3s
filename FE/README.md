# FE

Frontend starter built with React, Vite, React Router, Axios, and Bootstrap.

## Available scripts

- `npm run dev` - start the development server
- `npm run build` - create a production build
- `npm run lint` - run ESLint
- `npm run preview` - preview the production build locally

## Structure

- `src/components` - shared UI pieces
- `src/pages` - routed pages
- `src/services` - API clients and service helpers
- `src/layouts` - shared page shells
- `src/assets` - static assets
- `src/routes` - app routing

## Notes

The app ships with a responsive glass-style landing page and a reusable route layout. Configure `VITE_API_BASE_URL` in a `.env` file if you want to point Axios at a backend API.
